\hypertarget{md___users_matteobrini__documents__git_projects__pi__s_p_sim__r_e_a_d_m_e_autotoc_md3}{}\doxysection{ROOT output}\label{md___users_matteobrini__documents__git_projects__pi__s_p_sim__r_e_a_d_m_e_autotoc_md3}
As default, everytime a run is started, the program will make a file {\ttfamily output.\+root} in the {\ttfamily build} folder. The file contains the following TNtuples\+:
\begin{DoxyEnumerate}
\item FDet\+: information about the \$\textbackslash{}pi$^\wedge$-\/\$ in the first detector.
\begin{DoxyItemize}
\item f\+Events (int)\+: ID number of the event.
\item fX (double)\+: X position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\item fY (double)\+: Y position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\item fZ (double)\+: Z position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\end{DoxyItemize}
\item LH2\+: information about the process occurred at the end of the \$\textbackslash{}pi$^\wedge$-\/\$ life in the LH2 target.
\begin{DoxyItemize}
\item f\+Proc (string)\+: name of the occurred process.
\item fX (double)\+: final X position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\item fY (double)\+: final Y position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\item fZ (double)\+: final Z position of the \$\textbackslash{}pi$^\wedge$-\/\$.
\end{DoxyItemize}
\end{DoxyEnumerate}

The user can analyze the results contained in the file using the two scripts {\ttfamily plot\+\_\+3d.\+py} and {\ttfamily plot\+\_\+\+LH2.\+cc} in the project folder, or simply open a ROOT terminal to explore the file with a TBrowser.
